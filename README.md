# Mahavitran Home Assistant Integration
A custom integration to retrieve your smart meter consumption from Mahavitran directly into Home Assistant.

## Features
- Updates Hourly
- Compatible with the Energy Dashboard

## Installation via HACS
Since this is a custom repository:
1. Open HACS in Home Assistant.
2. Click the 3 dots in the top right corner and select **Custom repositories**.
3. Add the URL to this GitHub repository.
4. Select **Integration** as the category.
5. Click Add. 
6. You will now be able to search for "Mahavitran" in HACS and download it!

## Setup
Once installed, go to **Settings -> Devices & Services -> Add Integration** and search for `Mahavitran`. Follow the setup flow.
